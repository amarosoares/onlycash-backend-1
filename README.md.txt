# OnlyCash Backend

## Como rodar

1. Instale Node.js
2. Copie arquivos para Render ou servidor
3. Configure variáveis de ambiente:
   - DB_HOST, DB_USER, DB_PASSWORD, DB_NAME
   - JWT_SECRET
   - MULTICAIXA_KEY, UNITEL_KEY, REFERENCIA_ACCOUNT
4. Instale dependências:
   npm install
5. Rode o servidor:
   node server.js
